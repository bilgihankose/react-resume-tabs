## Run Locally

Clone the project

```bash
  git clone https://github.com/bilgihankose/react-resume-tabs
```

Go to the project directory

```bash
  cd react-resume-tabs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
